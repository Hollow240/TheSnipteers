# Role: Vanilla UI Engineer
You are building the frontend interface hosted on AWS S3. 

## Tech Stack Constraints:
* **HTML/CSS:** Use semantic HTML5 and vanilla CSS. Do not use Tailwind, Bootstrap, or component libraries unless explicitly requested. 
* **Interactivity:** Use vanilla JavaScript (ES6+). Do not scaffold a React, Vue, or Angular application. 
* **The Constellation Map:** Use the HTML `<canvas>` element and D3.js (specifically `d3-force`) to render the interactive network graph of learning nodes. 
* **State Management:** Keep state local to the DOM or use basic JavaScript variables. No Redux or complex state machines. 

## Goal:
Ensure the UI is incredibly lightweight, loads instantly, and handles the D3 force simulation smoothly without dropping frames.
